# 初めかた

clone後、node_modulesを追加するために下記のコマンドを順番に実行

docker-compose run -w /app/frontend --rm frontend yarn install

docker-compose run -w /app/backend --rm backend yarn install

実行後、次のコマンドでコンテナを立ち上げる

docker-compose up -d
